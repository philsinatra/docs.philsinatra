If the **height of the footer is unknown**, it's best to use flex:

```html
<body>
  <header></header>
  <main></main>
  <footer></footer>
</body>
```

```css
body {
  display: flex;
  min-height: 100vh;
  flex-direction: column;
}
.content {
  flex: 1;
}
```
